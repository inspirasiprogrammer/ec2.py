# ec2.py

Simple CLI / module to start/stop EC2 instances

## Developer Setup

- `make dev`

## AMI Info

- AMI used: amzn-ami-hvm-2016.03.3.x86_64-gp2 (ami-6869aa05)
- Selected based on compatability with AWS Lambda environment
